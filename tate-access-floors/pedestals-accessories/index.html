Product overview

The Tate pedestal range provides the support structure for Tate access flooring systems. Europed, Alpha V, Alpha V.5 and Alpha III options support different void heights, system types and load classifications.

Typical applications

  • Support systems for Tate raised access floors across commercial
  • technical and data centre applications

Technical summary

Zinc plated steel pedestal construction with multiple height ranges and support options.
